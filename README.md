# Tic-Tac-Toe Game in Python

## Project Overview
This project is a simple **console-based Tic-Tac-Toe game** built using Python and object-oriented programming.

Two players take turns entering moves on a 3x3 board. The game checks for:
- valid moves
- a winning condition
- a draw condition

The program also demonstrates the use of:
- classes
- objects
- methods
- loops
- condition checking
- random turn selection

---

## Features
- Two-player console gameplay
- Random selection of which player starts first
- Input validation for invalid or occupied positions
- Winner detection
- Draw detection
- Reward message for the winner
